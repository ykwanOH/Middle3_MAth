📁 파일 구성

이 학습 도구는 3개의 파일이 같은 폴더에 있어야 작동합니다:

📂 중3수학_학습도구/
  ├─ math_study_tool.html   ← 더블클릭으로 실행
  ├─ 신사고중학수학교과서.pdf   ← 본문 (243쪽)
  └─ 신사고중학수학답지.pdf   ← 정답 (41쪽)

🚀 실행 방법

방법 1: 로컬 파일로 직접 실행 (가장 간단)

  1. 위 3개 파일을 같은 폴더에 둡니다.
  2. math_study_tool.html을 더블클릭하면 Chrome에서 열립니다.
⚠ Chrome 보안 정책 주의
일부 환경에서는 file:// 프로토콜로 열 때 PDF.js가 차단될 수 있습니다.
그런 경우 아래 방법 2를 사용하세요.

방법 2: 로컬 서버로 실행 (안정적)

터미널/명령 프롬프트에서 파일이 있는 폴더로 이동한 후:

python3 -m http.server 8000  (또는 Python 2: python -m SimpleHTTPServer 8000)

그리고 브라우저에서 http://localhost:8000/math_study_tool.html 열기

방법 3: iPad에서 사용

iPad의 Chrome으로 접속하려면 PC에서 위 방법 2로 서버를 띄우고, 같은 Wi-Fi의 iPad에서 PC의 IP로 접속하세요. 예: http://192.168.0.10:8000/math_study_tool.html

🔑 Claude API 키 설정

  1. 도구를 열면 우상단에 ⚙ 설정 버튼이 있습니다.
  2. 발급받은 API 키 (sk-ant-...) 를 붙여넣고 저장합니다.
  3. 키는 이 브라우저의 localStorage에만 저장됩니다 (서버로 전송되지 않음).
  4. API 키 발급: console.anthropic.com

✨ 주요 기능

1. 목차 → PDF 페이지 점프 (좌측)

각 단원과 소단원을 클릭하면 해당 페이지가 즉시 표시됩니다.

2. 답지 보기 (우측 · 답지 탭)

현재 보고 있는 페이지의 답안 위치가 자동으로 매칭됩니다.
단, 「답지 보기」 버튼을 누르기 전까지 답은 보이지 않습니다.
학생이 먼저 문제를 푼 후 확인하는 방식입니다.

3. AI 추가 문제 (우측 · AI 탭)

현재 페이지의 단원에 맞는 추가 연습 문제를 Claude AI가 생성합니다.

  • 기본: 교과서 예제와 동일한 유형, 숫자만 변경
  • 응용: 같은 개념에 조건 추가, 2단계 추론
  • 심화: 단원 융합, 서술형, 상위권용

생성된 문제는 이동 가능한 팝업으로 표시되고, 풀이는 우측 패널에서 확인할 수 있습니다.
팝업을 닫아도 풀이는 우측 패널에 그대로 남아있습니다.

4. 오답 노트 (우측 · 오답 노트 탭)

틀린 문제를 메모와 함께 등록할 수 있습니다.
현재 페이지가 자동으로 기록되며, 나중에 클릭하면 그 페이지로 바로 이동합니다.
(브라우저 localStorage에 저장 — 같은 브라우저 한정)

⌨ 단축키

  • ← / → 또는 PgUp / PgDn : 페이지 이동

🐛 문제 해결

"교과서 PDF를 불러올 수 없습니다"

PDF 파일명이 정확한지, 같은 폴더에 있는지 확인하세요. 파일명이 다르면 설정에서 변경할 수 있습니다.

AI 문제 생성이 안 됨

  • API 키가 올바른지 확인 (sk-ant- 로 시작)
  • API 사용 한도 (Anthropic Console에서 확인)
  • 네트워크 연결 상태

iPad에서 PDF가 흐릿함

우측 상단의 확대(+) 버튼으로 배율을 높여보세요.
